# DevLabs

DevLabs is my personal portfolio website built with HTML, CSS, JavaScript, and React. It highlights my skills in UI/UX design, frontend development, and creative web solutions. The site is fully responsive and designed to provide a smooth user experience across devices.

Visit the live site: https://devsharma974-maker.github.io/DevLabs

## Development

Run the app locally:

```
npm install
npm start
```

## Deployment

This project uses `gh-pages` for deploying the production build to GitHub Pages. The `deploy` script runs `gh-pages -d build` and publishes the `build` folder to the `gh-pages` branch.
